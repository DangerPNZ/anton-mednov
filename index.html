Принципы и технологии:

Mobile first!

Сначала мобильные! Принцип, применяемый мной в создании адаптивных или отзывчивых веб-интерфейсов. Позволяет существенно рационализировать использование интернет-траффика мобильных устройств и быстродействие.
*Разработка адаптивного или отзывчивого проекта опциональна.

Progressive enhancement

Прогрессивное улучшение. Принцип создания страниц по пути от базовой функциональности и доступности контента к различным улучшениям.

Кроссбраузерность

Идентичное отображение разработанного проекта в различных браузерах и различных их версиях.
*Стандартно обеспечивается поддержка двух последних версий браузеров: Google chrome, Opera, Safari, Firefox, последней версии Internet Explorer.
**Иные случаи должны быть оговорены заказчиком перед началом разработки.

BEM

Блок-Элемент-Модификатор. Самодокументируемая методология, позволяющая оптимизировать процесс разработки документа. Позволяет упростить дальнейшую поддержку, расширение и повторное использование модулей.

Семантика

Семантичная вёрстка. Использование новых, продвинутых тегов для создания логической структуры html кода проекта.

Препроцессоры

Использование препроцессора less или sass. Позволяют повысить скорость разработки и упростить дальнейшую поддержку сайта.

Валидность

Законченный проект успешно проходит проверку программой-валидатором на отсутствие структурных ошибок.

Git

Использование системы создания разветвлённой структуры контроля версий.
*Позволяет заказчику наблюдать за ходом создания проекта. Желание следить за ходом разработки оговаривается отдельно.

Инструменты автоматизации

Использование Grunt или Gulp. Существенно ускоряет процесс разработки, обеспечивает лучшую оптимизацию конечного продукта. Позволяет создавать версию для удобной поддержки продукта и, максимально оптимизированную, для публикации.

Perfect Pixel

Вёрстка предоставленного заказчиком макета с высочайшей степенью точности соблюдения всех размеров, отступов и расстояний.
*Опционально.

ARIA

Расширенные интернет-приложения. Реализация дополнительной возможности использования проекта без мышки. Позволяет использовать сайт людям с ограниченным зрением, повышает удобство использования для любителей "горячих клавиш".
*Опционально.

SVG

Применение векторной графики для дизайна. При любом размере изображение не теряет качества, не появляется "пиксельности" и размытости. Также позволяет менять цвет изображения, или его частей при создании динамических эффектов.
*Опционально.

Ретинизация графики

Позволяет загружать на устройство то качество изображений, которое будет оптимально для данного дисплея. Позволяет существенно экономить интернет-траффик на дисплеях стандартной плотности и добиться отличного отображения на всех устройствах.
*Опционально.
